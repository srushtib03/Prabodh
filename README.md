**PRABODH** (Awakening) is an AI-powered educational platform designed to bridge the digital divide for students with disabilities ("Divyangjan"). It provides adaptive learning interfaces for **Blind** and **Deaf** students using Google Gemini AI.

## üöÄ Features

### 1. **Blind Mode (Visual Impairment)**
* **Voice-First Interface:** Students interact entirely via microphone and speakers.
* **High Contrast UI:** "Dark & Gold" theme for low-vision accessibility.
* **Speech-to-Text:** Converts spoken answers into text for AI analysis.

### 2. **Deaf Mode (Hearing Impairment)**
* **Visual Study Guides:** AI generates structured bullet points and emojis instead of audio.
* **Video Integration:** Automatically fetches relevant YouTube videos for visual reinforcement.
* **Visual Cues:** Uses color-coded notifications instead of sound alerts.

### 3. **Teacher Portal**
* Real-time dashboard to monitor student progress.
* Downloadable class reports (CSV/Excel).
